# Changelog

## 0.1.0

Initial scaffold of Forte AI Solutions marketing site.

- Next.js 14 with App Router and TypeScript strict mode
- Brand identity system: Cormorant, IBM Plex Sans, IBM Plex Mono typography
- Navy/brass color palette with noise texture and gradient treatments
- Home, About, Services, Contact pages with full copy
- Contact form with Zod validation and Upstash rate limiting
- Dynamic OG image generation
- JSON-LD structured data
- Responsive navbar with mobile drawer and focus trapping
- Framer Motion scroll animations with reduced motion support
- Security headers and CSP middleware
- llms.txt generation from constants
- Google Analytics and Vercel Analytics integration
